# SQL的ALL功能

## 功能

`ALL`运算符是一个逻辑运算符，它将单个值与子查询返回的单列值集进行比较。

**注意**

- 子查询只能返回一个字段，即只能返回单列。

## 语法

```sql
WHERE 列名称 比较运算符 ALL(子查询)
```

## 示例

已经有表table1：

| c1   | c2   | c3   |
| ---- | ---- | ---- |
| 1    | 11   | 111  |
| 2    | 22   | 222  |
| 3    | 33   | 333  |

已经有表table2：

| c1   | c2   |
| ---- | ---- |
| 1    | -1   |
| 2    | 0    |
| 3    | 1    |
| 4    | 2    |

查询要求，若table2.c2大于table1.c1中的所有值，则输出该table2的记录：

```sql
SELECT *
FROM table2
WHERE table2.c2 < ALL(SELECT c1 FROM table1);
```

| c1   | c2   |
| ---- | ---- |
| 1    | -1   |
| 2    | 0    |
