# IPv4地址的分类

## 概述

IP地址分为五类：A类、B类、C类、D类、E类。其中，A类、B类、C类是常用地址。

## 分类的标志

在二进制表示下，，每一类的IP地址都有固定的开头。

| 分类 | 二进制的开头 |
| ---- | ------------ |
| A类  | 0            |
| B类  | 10           |
| C类  | 110          |
| D类  | 1110         |
| E类  | 1111         |
