## 单模光纤和多模光纤比较

| 特性       | 单模光纤 | 多模光纤   |
| ---------- | -------- | ---------- |
| 距离       | 长       | 短         |
| 数据传输率 | 高       | 低         |
| 光源       | 激光     | 发光二极管 |
| 信号衰减   | 小       | 大         |
| 端结       | 较难     | 极易       |
| 造价       | 高       | 低         |
