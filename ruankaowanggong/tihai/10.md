编写算法，将一个节点类型为node的单链表逆序链接，即若为a1，a2，a3……，an，则逆序链接后变为an……，a3，a2，a1。

**答案**

void contrary(Node * &L)
{
    Node * P = L;
    L = NULL;
    while(p != NULL)
    {
        Node * q = p;
        q = p->next;
        q->next = L;
        L = q;
    }
}
