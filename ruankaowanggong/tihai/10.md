**题目** 

编写算法，将一个节点类型为Node的单链表逆序链接，即若为a1，a2，a3……，an，则逆序链接后变为an……，a3，a2，a1。

**答案**

```c++
void cojtrary(Node * L)
{
    Node * last = NULL;
    Node * next = NULL;
    while(L->next != NULL)
    {
        next = L->next;
        L->next = last;
        last = L;
        L = next;
    }
}
```
